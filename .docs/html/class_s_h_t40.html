uint8_t SHT40::init ( SHT40_unit_t  u)

Initialize SHT40 object.

Check provided external I2C operations handlers and sets temperature unit

Parameters
uDesired temperature unit.
Returns
SHT40_I2CH_R if I2C read handler is null.
SHT40_I2CH_W if I2C write handler is null.
SHT40_NOK if SN is 0x00000000 or 0xFFFFFFFF.
SHT40_OK if everything is OK.
